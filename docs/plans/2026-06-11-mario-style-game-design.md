# 超级小红帽 - 像素大冒险：设计文档

日期：2026-06-11
状态：已实现并通过浏览器实测

## 目标

豪华版超级玛丽风格网页平台跳跃游戏：双击 `index.html` 即玩，零构建、零外部资源依赖。

## 需求决策记录

| 决策点 | 选择 | 理由 |
|--------|------|------|
| 技术栈 | HTML5 + Phaser 3.80（本地 lib/） | 物理/动画/输入/场景管理开箱即用 |
| 规模 | 豪华版：5 关、4 种敌人、Boss 战、三形态、BGM | 用户选择 |
| 素材 | 代码生成原创像素画（非素材包） | file:// 协议禁止 XHR 加载外部图片；通用素材包无马里奥原型角色；规避任天堂版权 |
| 音频 | Web Audio 实时合成 8-bit 原创旋律 | 同上，且体积为零 |
| 操作 | 键盘 + 触屏虚拟按键 | 用户选择 |

## 架构

- `js/textures.js` — ASCII 像素图 + 调色板 → Canvas 纹理；换色生成地下/火焰变体
- `js/audio.js` — ChipAudio：方波/三角波旋律循环 + 16 种合成音效
- `js/levels.js` — LevelBuilder 按列生成 15 行字符网格；5 个关卡 + 3 套主题
- `js/entities/` — Player（小/大/火焰三形态）、Goomba、Koopa（壳机制）、Piranha、Boss（6 血、喷火、跳跃 AI）、道具与火球
- `js/scenes/` — Boot（生成纹理）→ Menu（标题/结算）→ Game（通用关卡播放器）+ UI（HUD/触屏按键，并行场景）

## 关键机制

- 字符地图：`#`地面 `B`砖 `?`金币 `M`道具 `!`星星 `{}[]`水管 `gkpX`敌人 `F`旗杆 `L`岩浆
- 受伤逐级降：火焰→大→小→死亡，附 2 秒无敌闪烁
- 敌人进入镜头范围才激活（防止跨图围攻出生点）
- 过关：旗杆滑落 + 时间奖励；最终关击败 Boss 通关
- 状态经由 registry（score/coins/lives/form/touch）跨场景共享
